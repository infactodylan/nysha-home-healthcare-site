## Cursor Cloud specific instructions

This is a Next.js 16 site using daisyUI 5 on Tailwind CSS v4, managed with npm. See `README.md` for standard dev commands.

**Key commands** (all from repo root):

| Task | Command |
|------|---------|
| Dev server | `npm run dev` (port 3000) |
| Build | `npm run build` |
| Lint | `npm run lint` |
| Tests | `npm test` |

**Non-obvious notes:**

- Tailwind CSS v4 uses `@plugin "daisyui"` in `src/app/globals.css` instead of a `tailwind.config.js`. Theme tokens live there too.
- `data-theme="light"` is set on the `<html>` element in `src/app/layout.tsx`; change it to switch daisyUI themes.
- Jest uses `ts-jest` + `jest-environment-jsdom`. Config is in `jest.config.js`; setup file is `jest.setup.ts`.
- The `AGENTS.md` header about Next.js breaking changes is auto-generated by `create-next-app` — keep it.
